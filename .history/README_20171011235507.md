# mo'c'k'S

#### 目录结构

    / 根目录
    |__ config 中间件配置
    |__ src 后端开发目录
    |   |__ router 路由
    |   |__ app.js 应用入口文件
    |__ bin node启动相关
    |__ doc 文档说明等
    |__ log 日志
    |__ public 前端开发目录
    |__ .eslintrc.js eslint脚本检查配置
    |__ package.json npm依赖管理


#### 相关命令

- 安装依赖

    npm i

- 启动项目

    npm run dev
